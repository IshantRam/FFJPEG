# FFjpeg

FFjpeg is a command line toolbox to manipulate, convert and much more to images.

## Run Locally

Clone the project

```bash
  git clone https://github.com/IshantSRam/FFJPEG
```

Go to the project directory

```bash
  cd FFJPEG
```

Compile

```bash
  make
```

Run

```bash
  make run
```
